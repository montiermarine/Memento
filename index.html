La ligne de commande

Gestion de documents

Pour créer un dossier je dois utiliser cette commande : mkdir

Pour créer un dossier et un sous dossier je dois utiliser cette commande : mkdir -p Nomddossier/nomdsousdossier

Pour créer un fichier je dois utiliser cette commande : touch

Pour déplacer un fichier je dois utiliser cette commande : mv

Pour afficher toute la liste de vos fichiers je dois utiliser cette commande : ls

Pour afficher le dossier courant je dois utiliser cette commande : pwd

Pour se déplacer dans un fichier je dois utiliser cette commande : cd
